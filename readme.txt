# Unity联网赛车游戏

这是一个基于Unity游戏引擎开发的联网赛车游戏项目，旨在提供一个参考和学习的资源。

## 项目概述

该项目包括服务器部分和客户端部分，所有代码均由本人独立完成。您可以使用本项目作为学习和参考，但请注意遵守适用法律和许可证。

## 功能特点

- 在线联网赛车游戏
- 客户端部分提供详细的UML类图
- 服务端部分提供详细的UML类图
- 提供游戏流程的时序图
- 目前实现如下功能：赛车操纵（前进、后退、转弯、刹车）、氮气系统（集气、氮气加速、氮气道具）、复位（自动复位、手动复位）、
	反向行驶检测、码表速度显示、圈数检测、每圈计时、小地图、过弯漂移、道具（氮气、加速带、传送门、弹射）、相机跟随、
	天气系统、开局倒计时、联网玩家位置同步、时间同步、结束排名、游戏UI、
	提供房间系统（可以自由创建房间、加入房间、退出房间）、房间提供聊天功能、
	选车界面提供视角旋转与缩放、支持拍照并保存到本地相册（需要存储权限）、
	提供两个地图场景，三个赛车模型，每个赛车支持多种纹理、渲染方式、颜色选取

## 如何使用

1. 下载工程文件到本地计算机。
2. 安装Unity游戏引擎。
3. 打开客户端部分的Unity项目文件。
4. 查看提供的UML类图以了解代码结构。
5. 查看时序图以理解游戏流程。
6、下载 `RacingGameServer` 文件夹，运行相关代码启动服务器服务（需要通过内网穿透工具映射出公网IP）
